About this Project and me

The SVG Generator was written by Simon Heimler in 2011. It was created as a project for the University of applied Science Augsburg in the subject Multimedia-Programming.

It is written in HTML5 with mainly Javascript (that means your Browser!) doing the work. There is a little PHP to allow the download of the generated SVG file.
